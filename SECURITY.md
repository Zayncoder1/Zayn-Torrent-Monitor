# Security Policy

## Supported Versions

We currently support the latest release.

| Version | Supported |
| ------- | --------- |
| 1.0.1   | Yes       |
| 1.0.0   | No        |

## Reporting a Vulnerability

Please report security issues using GitHub Security Advisories for this repository.

If you cannot use advisories, contact the repository owner via GitHub with:
- A clear description of the issue
- Steps to reproduce
- Impact (what could an attacker do?)
- Any relevant logs or screenshots

We will respond within 7 days with an initial assessment.
